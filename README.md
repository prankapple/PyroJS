# PyroJS
PyroJS is a library that adds explosions to your sites using gifs.

# Usage:
To import it to your html use : 
```html
<script src="https://rawcdn.githack.com/prankapple/PyroJS/f766d265db3a11d1a04d547be75a0bb0d5b5efaa/PyroJS.js"></script>
```
or
```html
<script src="https://prankapple.github.io/PyroJS/PyroJS.js"></script>
```
# How to use :

**Runs dynamite 1 time**
```javascript
PyroJS.dynamite(1);
```
**Runs TNT 1 time**
```javascript
PyroJS.tnt(1);
```
**Runs nuclear 1 time**
```javascript
PyroJS.nuclear(1);
```
