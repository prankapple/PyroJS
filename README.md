# PyroJS
PyroJS is a library that adds explosions to your sites using gifs.

# Usage:
To import it to your html use : 
```html
<script src="https://raw.githubusercontent.com/prankapple/PyroJS/refs/heads/main/PyroJS.js"></script>
```
# How to use :

**Runs dynamite 1 time**
```javascript
PyroJS.dynamite(1);
```
**Runs TNT 1 time**
```javascript
PyroJS.tnt(1);
```
**Runs nuclear 1 time**
```javascript
PyroJS.nuclear(1);
```
